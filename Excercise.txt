1. Explain what the terminal is in your own words.

2. What is the major difference between Git and Github?
Git is an open-source tool developers install locally to manage source code.
GitHub is an online service to which developers who use Git can connect and upload or download resources.

3. Why do we use Github?
GitHub is a repository, it allows your work to get out in front of the public.
